# AstroChart

> 🇰🇷 이 저장소는 **molpass가 포크한 사본**입니다. 영어 원문은 [README.en.md](./README.en.md)를 참고하세요.

![GitHub release](https://img.shields.io/github/v/release/AstroDraw/AstroChart?style=flat-square)

점성술에서 행성을 표시하는 SVG 차트를 생성하기 위한 무료 오픈소스 TypeScript 라이브러리입니다. 우주에서의 행성 위치 자체를 계산하지는 않습니다.

- 의존성 없는 순수 TypeScript 구현.
- SVG 그래픽.
- 테스트된 코드.

## 기여하기

기여는 언제나 환영합니다. 다양한 방식으로 참여할 수 있습니다:
 - [토론(discussions)](https://github.com/AstroDraw/AstroChart/discussions)을 시작하거나 참여하기
 - 열려 있는 이슈를 확인하거나 문서를 개선하기
 - 버그 제보나 개선 아이디어를 위해 [이슈](https://github.com/AstroDraw/AstroChart/issues) 열기
 - 버그 수정이나 새 기능에 대한 PR 열기. 작업 중복을 피하기 위해, 규모가 작지 않다면 먼저 이슈를 열어 논의하는 것이 좋습니다.

## 후원

AstroChart의 개발을 후원하고 싶으신가요? 다음과 같은 방법이 있습니다:

여러분의 프로젝트에서 사용 중이신가요? [여기에 댓글](https://github.com/AstroDraw/AstroChart/discussions/48)을 남겨 주시면, 이 라이브러리를 사용하는 멋진 프로젝트들을 함께 공유할 수 있습니다.

이 프로젝트를 다른 사람들과 공유하는 것도 좋은 후원 방법입니다.

또한, 회사라면 프로젝트 스폰서를 고려하거나 [커피 한 잔 사주기](https://ko-fi.com/afucher)도 가능합니다.
